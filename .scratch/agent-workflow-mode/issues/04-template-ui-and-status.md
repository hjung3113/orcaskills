# 04 — Template UI and status

Status: Completed

## Scope

Expose Agent Workflow creation and a constrained inspector/status presentation in the canvas UI.

## Done when

- Authors can create the template from the Studio without hand-authoring YAML.
- The UI shows fixed stages, per-stage profile selection, isolated CODEX boundary, local readiness, evidence state, and human-only release decision.
- Unsafe generic command/path editing is not exposed.

## Comments

- Implemented by the `✦ Agent Workflow` canvas creation action and template boundary panel.
